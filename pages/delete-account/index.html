본 안내는 출시일로부터 적용됩니다.

네컷모아는 앱 내에서 직접 계정 삭제(회원탈퇴)를 요청할 수 있습니다. 별도의 웹사이트 절차 없이, 아래 방법대로 진행해주세요.

삭제 방법

  1. 네컷모아 앱 실행 후 로그인
  2. 마이페이지 이동
  3. 회원탈퇴 메뉴 선택
  4. 탈퇴 사유 선택 (선택 항목)
  5. 안내 문구 확인 후 "네, 탈퇴할게요" 선택

탈퇴 신청과 동시에 계정 삭제가 처리되며, 별도의 대기 기간 없이 즉시 로그아웃됩니다.
